> Real-time simulation of ant colony behavior using ACO & Stigmergy algorithms.

🔴 [Live Demo](https://username.github.io/ant-colony/)

## ✨ Features
- 120 Workers + 20 Soldiers + Queen
- Pheromone diffusion & evaporation
- Real-time path optimization
- Interactive: add food, draw obstacles

## 🧠 How It Works
Ants communicate via **pheromones** — no central control.
Stronger paths get reinforced, weak ones fade. This is **Stigmergy**.

## 🚀 Run Locally
Just open `index.html` in your browser. No dependencies.

## 📚 Algorithms Used
- Ant Colony Optimization (ACO)
- Stigmergy
- Sensory Steering
